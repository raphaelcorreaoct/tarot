# Conteúdo SEO da Home

Cada `.md` aqui vira uma seção na home. Use **Markdown** comum:

- `**palavra**` — negrito (ótimo para palavras-chave)
- `## Subtítulo` — subcabeçalho
- `- item` ou `* item` — lista
- `[texto](url)` — link

**Modelo completo:** `o-que-e-tarot.md` — use como referência para os outros.

Arquivos que só têm comentários HTML (`<!-- ... -->`) não aparecem na página. Adicione conteúdo de verdade para a seção ser exibida.
